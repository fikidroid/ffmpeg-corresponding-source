# FFmpeg Corresponding Source

GPL complete corresponding source code for FFmpeg binaries bundled with **Songnara Studio** / **NEX AI**.

See [Releases](../../releases) for downloadable source archives.

## Why this repo exists

The applications above bundle unmodified FFmpeg binaries compiled with `--enable-gpl` (including libx264 and libx265). In accordance with the GNU General Public License, the complete corresponding source code for those exact builds is made available here, free of charge.
